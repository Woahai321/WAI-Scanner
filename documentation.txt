Steps to run

    - Make sure you unrar/unzip all the files first 
    - Make sure to have NodeJS installed in your system
    - Make sure to open this directory in Terminal / CMD
    - Run npm install
    - Run npm start

For figma file:
    - Open the figma and upload the given fig file directly into the file explorer
